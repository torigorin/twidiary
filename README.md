# MoneyLog — シンプル家計簿

みずほ銀行・楽天銀行・手持ち現金など、複数の貯金先の収入・支出をまとめて管理する個人利用向けのシンプルな家計簿アプリです。

## 使い方

`index.html` をブラウザで開くだけで動きます。サーバーもインストールも不要です。
GitHub Pages を有効にすると `https://<ユーザー名>.github.io/moneylog/` からも使えます。

## できること

- **口座別の残高管理** — みずほ銀行 / 楽天銀行 / 手持ち現金(名前の変更・追加・削除可)
- **収入・支出・振替の記録** — 振替は口座間の移動なので収支には含まれません
- **月ごとの貯蓄額(収入 − 支出)** — 今月の収支タイルと直近12ヶ月のグラフで確認
- **バックアップ** — JSONのエクスポート / インポート

## データの保存先

データはブラウザの localStorage のみに保存されます(外部送信なし)。
端末やブラウザを変えるときは「口座とデータの管理」からエクスポートして移行してください。
